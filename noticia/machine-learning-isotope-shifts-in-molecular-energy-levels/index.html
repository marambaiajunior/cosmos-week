Avanços recentes no uso de espectroscopia de correlação cruzada de alta resolução para detectar espécies moleculares em atmosferas de exoplanetas apresentam um novo desafio para a precisão das listas de linhas espectroscópicas de referência. Embora os isotopólogos parentais dos principais traçadores atmosféricos sejam frequentemente bem caracterizados, os isotopólogos menores, cruciais para o diagnóstico de histórias de formação e evolução planetária, sofrem com a escassez de dados experimentais, muitas vezes.

Avanços recentes no uso de espectroscopia de correlação cruzada de alta resolução (HRCCS) para detectar espécies moleculares em atmosferas de exoplanetas apresentam um novo desafio para a precisão das listas de linhas espectroscópicas de referência. Neste trabalho, uma estrutura abrangente de aprendizado de máquina é projetada para mitigar essas imprecisões, modelando os erros residuais do método de extrapolação de isotopólogos (IE) usado no projeto ExoMol.

É apresentado o desenvolvimento de uma nova arquitetura híbrida de aprendizagem por transferência com reconhecimento de moléculas que propaga com sucesso padrões de correção do sistema CO$_2$ rico em dados para o sistema de monóxido de carbono (CO) pobre em dados.

Esta abordagem de aprendizagem por transferência produz melhorias no MAE em mais de 93% das amostras de CO, demonstrando que os fatores de correção física relacionados à substituição isotópica podem ser generalizados em sistemas moleculares quimicamente relacionados. Listas de linhas atualizadas e melhoradas são apresentadas para 11 isotopólogos de CO$_2$ e os níveis de energia para estados excitados de isotopólogos de CO são previstos.

A metodologia estabelece um paradigma escalável e baseado em dados para refinar listas de linhas moleculares, ajudando a preencher a lacuna entre os cálculos teóricos e a precisão experimental. Tanto indivíduos como organizações que trabalham com arXivLabs abraçaram e aceitaram nossos valores de abertura, comunidade, excelência e privacidade dos dados do usuário.
