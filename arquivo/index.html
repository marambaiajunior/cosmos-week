Como usar o arquivo

O arquivo foi promovido de recurso escondido para área principal do portal. Ele serve para recuperar matérias antigas, seguir temas recorrentes e navegar pelo acervo com lógica editorial em vez de confiar só no acaso cronológico.

Leitura institucional

Para entender critérios de cobertura, rotulagem de fontes, uso de automação e correções, combine o arquivo com as páginas Sobre e Padrões.
