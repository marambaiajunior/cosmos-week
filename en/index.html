How to use this front page

  • Use the category rail to filter the current edition.
  • Use the archive when you want durable discovery across the whole corpus.
  • Use About and Standards when you need institutional clarity, not marketing fog.
  • Use the English routes for international circulation and navigation symmetry.

What changed in Phase 2

The main menu now points to real routes. The homepage explicitly introduces archive, methodology and institutional context. The bilingual structure is no longer buried in a toggle alone.
